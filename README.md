# LEG-DINO

Official implementation of **LEG-DINO: Local-Evidence-Governed DINO Fusion for Reliable Dense Prediction in Medical Image Segmentation**.

The complete source code, training scripts, configuration files, and reproducibility instructions will be released soon.

## Status

Code will be released soon.

## Task

Medical image segmentation using reliability-guided CNN–DINO feature fusion.

## Dataset

Experiments are conducted on ISIC2016, ISIC2018, PH2, FIVES and KAVISER dataset.
